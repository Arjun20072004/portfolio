# Portfolio

Lightweight static portfolio built with HTML, CSS, and JavaScript.

## Files

- [index.html](index.html) — main page
- [styles.css](styles.css) — styling
- [script.js](script.js) — small interactions

### Project gallery

- Thumbnails are in the `assets/` folder. Click "Preview" on a project to open the modal viewer.

## Run locally

Open `index.html` in your browser (double-click or use Live Server extension).

## Next steps

- Replace placeholder text and project links with your real content.
- Add images into an `assets/` folder and reference them from HTML.
- Deploy to GitHub Pages, Netlify, or Vercel for a live site.
 
## Try it

Open [index.html](index.html) in a browser. Click project "Preview" to see the modal viewer.
